MY WEATHER APP PROJECT.
Name: Uyo George Akpanama. Student ID: ALT/SOE/BAR/026/0170
This is lightweight Weather Application that fetches and displays live meteorological data for any city globally. I Built it using clean HTML5, CSS animations, and JavaScript while leveraging async/await patterns with API. This project consumes real-time telemetry from non-commercial public API service providing accessible weather data arrays globally without credential constraints. APP FEATURES: The App utilises a live featueres like: (1) Search tools: this instantly fetch real-time global conditions for any city worldwide. (2) 5-Day Forecast margin: It shows a five days alongside expected weather condition for any recognosed city around the world. (3) I have also included some animated graphics that react dynamically to conditions (4) The App also stores history of the last 5 locations etc. TECHNICAL COMPOSITION: (1) The App was constructed entirely on basic index.html structure. (2) The styling was done purely on basic CSS features. (3) JavaScript gave it the interactivity necessary for the standard. The App can be Run locally on a standard browser-supported web engines.
